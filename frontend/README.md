# Frontend

Frontend application for the decentralized voting platform.

Users can connect wallets, authenticate using SIWE, create proposals, and cast votes on-chain.

---

## Features

- Wallet connection
- SIWE authentication
- Proposal creation
- Vote casting
- Real-time blockchain interaction
- Responsive UI

---

## Tech Stack

- Next.js
- TypeScript
- TailwindCSS
- Wagmi
- RainbowKit

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

---

## Environment Variables

Create `.env` using `.env.example`.

---

## Wallet Features

- MetaMask support
- WalletConnect support
- Network detection
- Transaction state handling

---

## UI Features

- Responsive design
- Loading states
- Transaction feedback
- Error handling
